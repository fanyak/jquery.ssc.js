Social Shared Count Demo

Can get the count that your URL has been shared on social networking sites. This is dependent on jQuery or Zepto.

Download jquery.ssc.js

https://github.com/kuck1u/jquery.ssc.js

Set up

head inclusions ( jQuery ):
<link rel="stylesheet" type="text/css" href="assets/css/style.ssc.min.css" media="all">
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
<script src="assets/js/dist/jquery.ssc.min.js"></script>
head inclusions ( Zepto ):
<link rel="stylesheet" type="text/css" href="assets/css/style.ssc.min.css" media="all">
<script src="https://cdnjs.cloudflare.com/ajax/libs/zepto/1.1.6/zepto.min.js"></script>
<script src="assets/js/dist/jquery.ssc.min.js"></script>
JS call:
<script>
(function($){
$(".socialSharedCount").socialSharedCount({
    facebook: true,
    twitter: true,
    google: true,
    pocket: true,
    hatena: true,
    linkedin: true,
    twitterVia: 'kuck1u',
    twitterRelated: 'visualive_jp'
});
})(jQuery);
</script>
HTML constructor:
<div class="socialSharedCount" data-ssc-url="https://www.google.com/" data-ssc-title="Google"></div>
